Privacy Policy for Sliding It Translation Extension

Last updated: 2024/12/23

Introduction

This Privacy Policy describes how Sliding It Translation Extension ("we", "our", or "the extension") collects, uses, and protects information when you use our browser extension. We are committed to ensuring your privacy and protecting any data collected through the extension.

Information Collection and Use

The extension collects the following limited information: - Text selected by users for translation - User preferences (target language and automatic translation settings)

This information is collected solely for the purpose of: - Providing instant translation services - Maintaining your preferred translation settings - Improving the translation service quality

Data Storage

  • User preferences are stored locally in your browser using Chrome's storage API
  • Translation settings are synchronized across your devices when you're signed into Chrome
  • No personal information or translation history is permanently stored

Data Transmission

  • Selected text is transmitted to our translation API (sliding-api.youngzm.com) for processing
  • All data transmission is encrypted using HTTPS
  • Data is transmitted only when you actively select text for translation

Third-Party Services

The extension uses: - Marked.js library for rendering translation results - Our dedicated translation API service

These services are essential for core functionality and do not collect additional user data.

Data Protection

We implement security measures to protect your data: - Encrypted data transmission - No persistent storage of translated content - No collection of browsing history or personal information

User Rights

You have the right to: - Disable the extension at any time - Clear stored preferences through Chrome's extension settings - Control when translations occur through extension settings

Changes to Privacy Policy

We may update this Privacy Policy periodically. Significant changes will be notified through the extension's update notes or website.

Contact Information

For privacy-related questions or concerns, please contact us at: youngzm339@outlook.com

Compliance

This extension complies with: - Chrome Web Store Developer Program Policies - General data protection and privacy regulations

By using the Sliding It Translation Extension, you agree to the collection and use of information in accordance with this policy.
